# Análise de dados de vendas com SQL

## Visão geral do projeto
Este projeto analisa dados de vendas usando MySQL para extrair informações relevantes para o negócio.

## Tools
- MySQL
- SQL
- Data Analysis

## Análises realizadas
- Análise de receita
- Cálculo do lucro
- Desempenho regional
- Tendências de vendas
- Segmentação de clientes

## Key Insights
- Regiões de melhor desempenho identificadas
- Canais de vendas avaliados
- Comportamento do cliente analisado

## Dataset
Conjunto de dados sintéticos de vendas para fins de análise.
